Functions

Function is a sequence of program instructions that performs a specific task, packaged as a unit. When we create a function, we can name the arguments it takes (but we don't have to). Arguments are variables that we give to the function so that it can operate on them (generally, a function cannot access objects created outside it due to variable scopes, which I will discuss in a moment).

A function can return objects (variables, numbers, arithmetic operations, etc.). When it does, the line that calls a function is equal to the object it returns. To see what the function returned, we have to print the "calling" of it in the console (using print). To call a function (run it), we write its name and a parenthesis. If it takes any arguments, we write them inside the parenthesis.


def function():
	print("Function")

def addition(x, y):
	return x + y

function()
x = 2
y = 4
print(addition(x, y))

If we want to leave a function empty for a while, we can write pass as its content so as to prevent a syntax error. This keyword also works with loops, conditional statements, etc.

Variable scopes

Not every variable is accessible in every part of the program. Variables created in the main block (outside of any functions), are called global variables. They are accessible in all functions and instructions (under one condition, which I will discuss in a moment). Variables that are created inside the same block we want to use them from are called local variables, and are always available.

To get access to a global variable inside a function, we "import" this variable to the function with the global keyword. In Python, we also have a nonlocal variable scope. Nonlocal variables are variables that are declared within nested functions. We can "import" a nonlocal variable the same way as the global variable, but we use the nonlocal keyword.

Now that we know what variable scopes are, we can see that in the function example, even if we change the values of the x and y variables in the addition() function, their values in the main block will not change. These variables, when in the addition() function, are local variables whose values have been imported as arguments and are then returned to the main block.


x = 3

def func1():
	y = 4
	def func2():
		global x
		nonlocal y
		x = 5
		y = 6
	func2()
print(x, y)

func1()

Be mindful that nesting variables in the context of variable scopes, is only relevant for functions, not for loops and conditional statements.

Arguments


# Returning a value multiplicated by a given number of times	
list1 = [1, 2]
def function(x):
	return list1 * x
print(function(3))
		
# Setting default arguments (we can, but don't have to give them)
def func1(x, y = 1):
	return x + y
print(func1(1))
print(func1(1, 2))

# Telling another programmer what types the arguments should be (it doesn't obligate to do anything)
def func2(x:int, y:int):
	return x + y
print(func2(1, 2))

# Telling another programmer what type the returned value should be (it doesn't obligate to do anything)
def func3(x, y) -> str:
	return str(x + y)
print(func3(1, 2))

Recursive functions

A recursive function is a function that calls itself. Recursion is often used in various algorithms.


def function():
	print("x")
	function()
function()
