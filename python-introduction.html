Introduction

Python IDLE

You can download Python from this website. In Python IDLE we work on two windows. After launching, the Shell Window will appear. The results of our programs will be displayed in it. To create a new file, click File / New File (Ctrl + N). To open an existing file, click File / Open (Ctrl + O). We can't open .py files by just clicking on them - we have to open them in the Shell Window. In the other window, which will appear after creating or opening a file, we will write our code. We will run it by clicking Run, and in this tab, Run Module.

PyCharm

You can download Python from this website. If you want PyCharm to work, you must install Python IDLE first because it is only a virtual environment. We are installing it because it's easier to use than Python IDLE (e.g., it suggests correct syntax). In PyCharm, we work on one window. The program results print at the bottom of the screen after you click Run / Run. To create a new file, click File / New File (we type the file name in there), and to create a new project (it can consist of many .py files), click File / New Project. If you have closed some project file and you want to open it, click on the tab with the project name (on the left side) and unfold it. Every project and its files are here: C:\Users\[Username]\PycharmProjects\. Instead of PyCharm, you can also use VS Code, Atom, Spyder, Sublime Text, Vim, or Jupyter.
