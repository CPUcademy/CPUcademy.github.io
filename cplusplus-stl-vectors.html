STL - vectors

STL stands for Standard Template Library. Vectors are dynamic arrays, which means we do not have to know their size at the beginning (but we have to determine the type of the elements). It works because the memory is reserved in advance based on how much elements we added in the past. That reserved in advance memory is the vector's capacity, and its size is how much of this memory is used. We have to include the <vector> library for all of this to work.

The push_back() command will add an element at the end of a vector. If we want to use all of the vector's features, e.g., assigning values at the beginning (vector<int> numbers {1, 2, 3, 4, 5};), we have to enable C++ 11 in the editor. In Dev C++, we can do that here: Tools / Compiler Options / Settings / Code Generation / Language Standard (-std) - GNU C++11.


#include <iostream>
#include <vector>
using namespace std;

int main()
{	
	vector<int> integers;
	
	integers.push_back(2);
	integers.push_back(4);
	
	cout << integers[0] << endl;
	cout << integers.at(0) << endl; // the same as integer[0]
	cout << integers[1] << endl;
	cout << integers.front() << endl; // the first value
	cout << integers.back() << endl; // the last value
	cout << "Size: " << integers.size() << endl;
	cout << "Capacity: " << integers.capacity() << endl;
	
	return 0;
}

If you want to make a vector of arrays, you have to import the <array> library and write: vector<array<string, 5>> data;. In this example, each array in the vector has five elements. Of course, you can also nest vectors;

Iterators

An iterator is an object pointing at a particular position inside a container (a vector). We use it to get to certain vector's elements. We create iterators with this pattern: "type_of_the_container::iterator name;". Not all elements inside a container are in a row inside of the memory (this was the case with arrays).


vector<int> numbers {10, 20, 30, 40, 50};

vector<int>::iterator i = numbers.begin(); // it points at the first element of the "numbers" vector 

cout << *i << endl; // it isn't a pointer - it is an object
i += 2;
cout << *i << endl;
									

while(i != numbers.end())
{
	cout << *i << endl;
	i++;
}
									

for(vector<int>::iterator i = numbers.begin(); i != numbers.end(); i++;)
{
	cout << *i << endl;
}

auto type

auto is a data type that decides for us which type is the variable based on its value. We can check the variable's type with the typeid(x).name() command from the <typeinfo> library.


auto x = 5;
auto y = 5.5;
auto z = "z";

for(auto i = numbers.begin(); i != numbers.end(); i++;)
{
	cout << *i << endl;
}

Enhanced for

Enhanced for is a type of the for loop with the syntax shown below. If we add an ampersand after the auto keyword, we will be able to edit the values of a vector inside the loop.


for(auto x : numbers) // where to save the values : from where to read the values
{
	cout << x << endl;
}

for(auto& x : numbers)
{
	x *= 10;
	cout << x << endl;
}
