Loops

The for loop

The for loop is a repetition loop. It repeats given instructions a certain number of times (of course, this number can be typed in as a variable). In the example below, the loop will work from 0 to 6 (6 times) for the x variable (this variable didn't have to be created earlier). This variable is called an iterator. We don't have to add the zero before the number of times because it will add itself, but we can write, e.g., from 1 to 7, and the result will be the same. As long as we don't use nested loops (a loop in a loop), we can use the same variable names as iterators in each of the loops because their value resets during reinitialization.


for x in range(0, 6):
	print("Hello World!")
	
for x in range(6):
	print("Hello World!")
	
x = 6
for y in range(x):
	print("Hello World!")

We can add a "step" argument to a loop. If the range is from zero to ten, and this argument equals two, the loop will execute five times because it will "jump" every two.


for x in range(0, 10, 2):
	print("Hello World!")

In the following example, the first loop will execute the other four times (and the second one will execute two times). That will give us eight repetitions overall. We can also nest a while loop in a for loop, etc.


for x in range(4):
	for y in range(2):
		print("Hello World!")
