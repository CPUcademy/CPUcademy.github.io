Basic libraries, modules, and pip

A library is a collection of related modules. To add a library, we use the import command, e.g., import random. However, we would have to write the library's name before every command that comes from it. If we don't want to do this, instead of import ..., we can use the from ... import * command.

The random library

The random library is used to generate random numbers. The numbers generated using this library are pseudorandom, which means they are drawn according to a certain pattern. The initial value used by the generator is called a seed. If we know the value of the seed, we can predict the rest of the numbers produced by the generator because, for a given seed, the same numbers will always be generated (in the same order). The seed can be set with the random.seed(value) command.

Use the import random command at the beginning of every example.


# Drawing integers (we determine the range in the arguments - start, stop)
x = random.randint(1, 1000)
print(x)
									

# Drawing integers but with a third argument - step (how many numbers to "jump" while drawing)
x = random.randrange(1, 10, 2) # it can draw these numbers: 1, 3, 5, 7, 9 
print(x)
									

# Shuffling the list's elements
x = ["abc", "def", "ghi", 1, 2, 3, 1.1, 2.2, 3.3]
random.shuffle(x)
print(x)
									

# Drawing from the list's elements
x = ["abc", "def", "ghi", 1, 2, 3, 1.1, 2.2, 3.3]
print(random.choice(x))
									

# Drawing float numbers (we determine the range in the arguments - start, stop)
x = random.uniform(20, 60)
print(x)

The math library

The math library provides more complex mathematical operations.


import math
x = 16
y = [1, 2, 3, 4]

print(math.ceil(x)) # rounding up a number
print(math.floor(x)) # rounding down a number
print(math.fsum(y)) # adding up all the list's elements
print(math.gcd(4, 8)) # calculating the Greatest Common Divisor
print(math.sqrt(x)) # calculating the square root of a number
print(math.pow(x, 2)) # raising a number to a given power
print(math.isclose(2, 4)) # checking whether the given numbers are close to each other (in hundredths)
print(math.log(9, 3)) # calculating the power to which "number 2" must be raised to get "number 1"
print(math.log2(64)) # calculating the power to which 2 must be raised to get a given number
print(math.pi) # printing the PI number
print(math.e) # printing the E number

The time and datetime libraries

These libraries are used to handle time.


import time

# Stopping the program for a time specified in seconds
time.sleep(0.5)

# Printing the time
print(time.strftime("Time: %H:%M:%S"))
# %Y - year
# %B - month (a string)
# %m - month (a number)
# %d - day
# %H - hour
# %M - minute
# %S - second
									

import datetime
time = datetime.datetime.now()

print(time) # time and date
print(time.second)
print(time.minute)
print(time.hour)
print(time.day)
print(time.month)
print(time.year)

The sys and os libraries

The sys and os libraries handle system operations.


import sys
for x in range(4):
		print("Python")
		sys.exit() # shutting down the program
									

import os
if os.path.exists("file.txt"): # checking whether a given file or directory exists
		print("The file exists")
		os.remove("[the_path]/file.txt") # deleting a given file or directory

Pip

At this level, everyone has probably already encountered this command. It is the official and default package management system for the Python language environment, using a dedicated repository called Python Package Index or other remote and local repositories. The list of modules that can be downloaded with pip is available at the pypi.org website. The pip module can be found in this folder: C:\Users\[username]\AppData\Local\Programs\Python\Python311\Scripts. To use it, we must move to this location in the CMD console using the cd command. Then, we can execute, for example, the pip install ... command, which will install a library that is not installed by default. If we want to update a library, we have to type: pip install --upgrade .... To update pip itself, we would have to use this command: python.exe -m pip install --upgrade pip.
