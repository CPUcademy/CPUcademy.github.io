".py" to ".exe" (works with every library)

Jeśli chcesz, żeby twoje działania zgadzały się z moimi, zainstaluj edytor PyCharm.

".py" na ".exe"

1. Wchodzimy w CMD i przenosimy się do projektów PyCharm (cd C:\Users\[nazwaUżytkownika]\PycharmProjects\[nazwaProjektu]\venv\Scripts).

2. Instalujemy bibliotekę auto-py-to-exe tą komendą: pip install auto-py-to-exe.

3. Po jej zainstalowaniu, wpisujemy do CMD: auto-py-to-exe i czekamy chwilę, aż wyświetli się nam graficzne okienko.

4. Wpisujemy ścieżkę do pliku, który chcemy przekonwertować, zaznaczamy One File i Window Based (hide the console). Następnie wybieramy ikonę (jeśli ją posiadamy) i klikamy CONVERT .PY TO .EXE (nie wybieramy potrzebnych plików w Additional Files, nawet jeśli takie są).

5. Gotową aplikację .exe możemy znaleźć w folderze wyjściowym (po zakończeniu działań pojawi się przycisk open output folder).

Pygame na ".exe"

1. Wchodzimy w CMD i przenosimy się do projektów PyCharm (cd C:\Users\[nazwaUżytkownika]\PycharmProjects\[nazwaProjektu]\venv\Scripts).

2. Instalujemy Nuitkę tą komendą: python -m pip install -U nuitka.

3. Po jej zainstalowaniu, wpisujemy do CMD komendę: python -m nuitka C:\Users\[nazwaUżytkownika]\PycharmProjects\[nazwaProjektu]\[nazwaPliku]. Jeśli mamy ikonę, którą chcielibyśmy dodać, po słowie nuitka dopisujemy --windows-icon-from-ico=[ścieżkaDoPliku]icon.ico.

4. Po chwili wszystkie procesy się zakończą, a nasz plik .exe znajdziemy w tej lokalizacji: C:\Users\[nazwaUżytkownika]\PycharmProjects\[nazwaProjektu]\venv\Scripts.
