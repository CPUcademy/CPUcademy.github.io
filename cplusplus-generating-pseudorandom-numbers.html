Generating pseudorandom numbers

We can't generate truly random numbers because a computer doesn't have the element of randomness. We can only generate pseudorandom numbers, which means they are generated based on a seed. For a certain seed the generator will always draw the same number. In C++, we have to set the seed manually after every time we draw a number because otherwise it will be the same. We can use time to achieve this because it is in constant movement.

To generate random numbers, we use the <stdlib.h> library. We can set a seed with the srand() command (its argument is the seed value) and generate a number with the rand() command (it doesn't take any arguments). If we want to set a range from which we can draw numbers, we can write, e.g., rand() % 3, which will set the range of numbers to 0-2.

To operate on time, we use the <time.h> library. We can use the time() method to return the current calendar time as a single arithmetic value (this method takes a pointer as an argument). This arithmetic value is the number of seconds elapsed since 00:00:00 hours, GMT (Greenwich Mean Time), January 1, 1970. If we give NULL as this argument, it will only return the value and not save it in a pointer (NULL is an object that is used to indicate the absence of a value).


#include <iostream>
#include <stdlib.h>
#include <time.h>
using namespace std;

int *lotto(int total_balls, int balls_to_allot)
{
	if(total_balls < balls_to_allot) return NULL;
	srand(time(NULL));
	int *balls = new int[balls_to_allot];
	
	for(int i = 0; i < balls_to_allot; i++)
	{
		balls[i] = rand() % total_balls + 1; // the range will be 1-49
		for(int j = 0; j <= i; j++)
		{
			if(balls[i] == balls[j] && i != j)
			{
				i--;
				break;
			}
		}
	}
	return balls;
}

int main()
{
	srand(time(NULL));
	int liczba = rand() % 3;
	cout << liczba << endl;
	
	int *balls1 = lotto(49, 6);
	
	for(int i = 0; i < 6; i++)
	{
		cout << "ball [" << i + 1 << "]  = " << balls1[i] << endl;
	}
	delete[] balls1;
	
	return 0;
}
