Kotlin

Kotlin is a modern, concise, and safe programming language that is fully interoperable with Java. This means anything we can freely mix Java and Kotlin code within the same project. You can call Kotlin functions from Java, use Java classes in Kotlin, and even pass variables and objects between the two languages seamlessly. It runs on the JVM and is officially supported for Android development, while reducing boilerplate code and adding powerful features like null safety, type inference, and functional programming constructs.

Variables and type inference

Kotlin uses val for immutable variables (cannot be reassigned) and var for mutable variables (can be reassigned). You can explicitly specify a variable’s type using :, or let Kotlin infer it automatically from the value.


fun main() {
    val username: String = "Alice" // immutable variable of type String"
    var score: Int = 42 // mutable variable of type Int"
    
    val level = 3 // type inferred as Int"
    var isActive = true // type inferred as Boolean"
    
    println("$username has score $score and level $level, active: $isActive")
}

Explanation:

  • val username: String = "Alice" declares an immutable variable of type String.
  • var score: Int = 42 declares a mutable variable of type Int that can change later.
  • If you omit the type, e.g., val level = 3, Kotlin automatically infers it from the value.

Conditions and loops

Kotlin supports if, when (like switch), and standard loops: for, while, and do-while.


fun main() {
    val numbers = listOf(1, 2, 3, 4, 5)

    for (number in numbers) {
        println(number)
    }

    val x = 3
    val message = when (x) {
        1 -> "One"
        2 -> "Two"
        3 -> "Three"
        else -> "Unknown"
    }
    println(message)
}

Functions and default arguments

Kotlin functions are declared with fun. You can use default arguments, named arguments, and single-expression functions.


fun greet(name: String, greeting: String = "Hello") {
    println("$greeting, $name!")
}

fun main() {
    greet("Alice") // uses default greeting"
    greet(name = "Bob", greeting = "Hi") // named argument"
}

Classes and inheritance

Classes are declared with class. By default, classes and methods are final and cannot be inherited or overridden. Use open to allow inheritance and override for overriding methods.


open class Animal {
    open fun voice() {
        println("Some generic sound")
    }
}

class Cat : Animal() {
    @Override
    override fun voice() {
        println("Meow!")
    }
}

fun main() {
    val cat = Cat()
    cat.voice()
}

Null safety

Kotlin distinguishes nullable and non-nullable types. Use ? to allow nulls and ?. for safe calls, or ?: (elvis) for default values.


fun main() {
    var name: String? = "Kotlin"
    println(name?.length) // safe call"

    name = null
    println(name?.length ?: 0) // elvis operator, returns 0 if null"
}

Android example: button click listener

Here’s an example of setting up a simple click listener in Kotlin for an Android Button:


import android.os.Bundle
import android.widget.Button
import androidx.appcompat.app.AppCompatActivity

class MainActivity : AppCompatActivity() {
    @Override
    fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_main)

        val button: Button = findViewById(R.id.myButton)
        button.setOnClickListener {
            println("Button clicked!")
        }
    }
}

Additional Android example: lambda with RecyclerView adapter

Kotlin allows concise lambdas for click handling in adapters:


class ItemAdapter(private val items: List, 
                  private val onItemClick: (String) -> Unit) 
                  : RecyclerView.Adapter<ItemAdapter.ViewHolder>() {

    inner class ViewHolder(val button: Button) : RecyclerView.ViewHolder(button)

    @Override
    fun onCreateViewHolder(parent: ViewGroup, viewType: Int) : ViewHolder {
        val button = Button(parent.context)
        return ViewHolder(button)
    }

    @Override
    fun onBindViewHolder(holder: ViewHolder, position: Int) {
        holder.button.text = items[position]
        holder.button.setOnClickListener { onItemClick(items[position]) }
    }

    @Override
    fun getItemCount() = items.size
}
