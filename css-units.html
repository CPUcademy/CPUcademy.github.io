Units

Podstawową jednostką w CSS jest pixel (px), ale istnieją jeszcze inne. Dzielimy je na absolutne i relatywne.

Piksel to najmniejszy, niepodzielny, dający się wyróżnić element wyświetlanego obrazu. Wymiar piksela można spróbować odnieść do np. mm, dopiero znając DPI (ang. dots per inch) wyświetlacza. DPI wyświetlacza to liczba punktów przypadających na cal (zależy od rozdzielczości). Im urządzenie jest mniejsze, tym większe powinno być DPI.

Rozmiar piksela w CSS (emulowanego na wyświetlaczu) to 1/96 cala. Antyaliasing to proces dodawania pikseli do rysunku, które zmniejszają kontrast pomiędzy wyrenderowanym obiektem i jego tłem. Renderowany piksel nie jest równoznaczny z pikselem systemowym.

Procent to zawsze jednostka relatywna, ale zachowa się jak wartość niezależna, gdy ustawimy relację wobec jednostki absolutnej. Procenty mogą więc określać zarówno wymiar dynamicznie zmienny jak i wartość absolutną (zależy to od tego, co ustawiono u "rodzica" w kaskadzie).

Element "rootowy" w dokumencie html to znacznik <html> lub pseudoklasa :root.

em:

- Stylizowane proporcją.

- Sam wybierz rodzica, względem którego ustawisz rozmiar.

rem:

- Również stylizowane proporcją.

- Rozmiar ustawisz wobec "roota".

Viewport to obszar okna przeglądarki. W mowie potocznej często używamy pojęć: viewport, płótno, okno, obszar i canvas zamiennie (równoważnie). Jednak dokładnie mówiąc, to viewport definiujemy jako obszar, w którym oglądamy witrynę, lecz bez obramowania okna oraz możliwych suwaków. Natomiast półtno (canvas) może być większe niż viewport - wówczas pojawiają się suwaki. Podsumowując, płótno to cały obszar witryny, i to nawet taki, który nie mieści się w viewporcie, a powiedzenie po prostu "okno" teoretycznie powinno się odnosić tak do viewportu, jak również zakładek i paska adresu przeglądarki.

vw - viewport width.

vh - viewport height.

Jednostek vw i vh możemy używać jako width i height niezależnie od siebie (vw możemy ustawić jako height i na odwrót - nie ma ograniczeń).

Istnieją jeszcze jednostki takie jak vmax i vmin. Jeśli oś x jest większa (na ekranie) to vmax jest osią x, a jeśli np.: na telefonie oś y jest większa, to vmax jest osią y i analogicznie vmin.

calc() - funkcja przeprowadzająca obliczenia pomiędzy różnymi jednostkami w CSS np.: width: calc(100% - 200px);. Ta funkcja nie zadziała, jeśli po obu stronach operatora (w tym przypadku -) nie będzie spacji! Dostępne są operatory: +, -, * i /. Mnożenie i dzielenie nie wymaga tych spacji.
