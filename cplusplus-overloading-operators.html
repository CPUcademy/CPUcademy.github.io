Overloading operators

Overloading an operator means overwriting its functionalities so it can work with, e.g., our class's objects. Remember that we cannot simply display our objects with cout because the << operator is not defined for them. To display our objects, we would have to overload this operator. The example below shows how to overload the == operator so it can compare our objects.


#include <iostream>
using namespace std;

class Comparison
{
    int x, y;
    public:
        Comparison(int x, int y)
        {
            this -> x = x;
            this -> y = y;
        }
        ~Comparison(){}
        
        int operator==(Comparison obj)
        {
            if(this -> x == x && this -> y == y)
                return 1;
            return 0;
        }
};

int main()
{
    Comparison x(10, 20);
    Comparison y(10, 20);
    
    if(x == y)
        cout << "a and b are the same" << endl;
    
    return 0;
}
