Moviepy

Biblioteka Moviepy służy do edycji filmów i dźwięków. Aby można było wyświetlać filmy i odtwarzać dźwięki, należy zainstalować także bibliotekę pygame (bez niej można je edytować, ale nie uruchamiać). Ta biblioteka wymaga dodatkowego zainstalowania. Aby dowiedzieć się, jak ją zainstalować, przejdź tutaj.

Obsługa filmów

Odtwarzanie filmu

Komenda preview() pochodzi z biblioteki pygame i służy do wyświetlania filmów (wyświetla się on wtedy na cały ekran i nie można go zatrzymać).


									from moviepy.editor import *
									clip = VideoFileClip("intro.mp4")
									clip.preview()

Przekręcanie filmu

Komenda rotate() obróci film o wybrany stopień, a write_videofile() wyeksportuje go.


									from moviepy.editor import *
									clip = VideoFileClip("intro.mp4")
									flipped = clip.rotate(35)
									flipped.write_videofile("intro-flipped.mp4")

Przycinanie filmu

Poniższy kod skróci film o połowę.


									from moviepy.editor import *
									clip = VideoFileClip("intro.mp4")
									czas = clip.duration
									shortened = clip.subclip(0, czas/2)
									shortened.write_videofile("intro-shortened.mp4")

Przyspieszanie filmu

Poniższy kod przyspieszy film 2x.


									from moviepy.editor import *
									clip = VideoFileClip("intro.mp4").fx(vfx.speedx, 2)
									clip.write_videofile("intro-accelerated.mp4")

Łączenie filmów

Poniższy kod sprawi, że dwa filmy scalą się w jeden.


									from moviepy.editor import *
									clip1 = VideoFileClip("intro.mp4")
									clip2 = VideoFileClip("outro.mp4")
									mix = concatenate_videoclips([clip1, clip2])
									mix.write_videofile("mix.mp4")

Mieszanie filmów

Poniższy kod stworzy cztery kopie filmu, ale do każdej zastosuje inny efekt, a następnie złączy je w "kolaż".


									from moviepy.editor import *
									clip1 = VideoFileClip("intro.mp4").margin(10)
									clip2 = clip1.fx(vfx.mirror_x)
									clip3 = clip1.fx(vfx.mirror_y)
									clip4 = clip1.resize(0.60)
									clip = clips_array([clip1, clip2], [clip3, clip4])
									clip.resize(width=480).write_videofile("intro-collage.mp4")

.mp4 na .gif

Poniższy kod skonwertuje plik .mp4 na plik .gif.


									from moviepy.editor import *
									clip = VideoFileClip("intro.mp4")
									clip.write_gif("intro-gif.gif")

Obsługa dźwięków

Odtwarzanie dźwięku

Komenda preview() służy także do odtwarzania dźwięków (tego dźwięku nie można zatrzymać).


									from moviepy.editor import *
									clip = AudioFileClip("audio.mp3")
									clip.preview()

Podmienianie dźwięku w filmie

Poniższy kod podmieni dźwięk w filmie intro.mp4 na audio.mp3 i wyeksportuje film ze zmienionym dźwiękiem.


									from moviepy.editor import *
									videoclip = VideoFileClip("intro.mp4")
									audioclip = AudioFileClip("audio.mp3")
									videoclip2 = videoclip.set_audio(audioclip)
									videoclip2.write_videofile("intro2.mp4")

Przyspieszanie dźwięku

Poniższy kod przyspieszy dźwięk 2x.


									from moviepy.editor import *
									clip = AudioFileClip("audio.mp3").fx(vfx.speedx, 2)
									clip.write_audiofile("audio-accelerated.mp3")

Łączenie dźwięków

Poniższy kod złączy trzy pliki audio.


									from moviepy.editor import *
									clip1 = AudioFileClip("audio1.mp3")
									clip2 = AudioFileClip("audio2.mp3")
									clip3 = AudioFileClip("audio3.mp3")
									mix = concatenate_audioclips([clip1, clip2, clip3])
									mix.write_audiofile("mix.mp3")
