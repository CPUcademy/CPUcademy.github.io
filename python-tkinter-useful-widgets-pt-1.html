Useful widgets pt. 1

Text

Dzięki widżetowi Text możemy wyświetlać tekst, który można zaznaczyć i skopiować (nie można tak zrobić z Label).


                                    import tkinter
                                    window = tkinter.Tk()
                                    myText = tkinter.Text(window, width=30, height=2)
                                    myText.pack()
                                    myText.insert(tkinter.END, "Wonsz\nPython")
                                    window.mainloop()

Żeby zczytać tekst wpisany przez użytkownika do tego widżetu, zapisując każdą linijkę jako inny element listy, używamy tego kodu:


                                    import tkinter
                                    window = tkinter.Tk()

                                    def get():
                                            lista = []
                                            text = str(myText.get(1.0, tkinter.END))
                                            latestEnter = 0
                                            for x, y in enumerate(text):
                                                    if y == "\n":
                                                            toAppend = text[latestEnter:x]
                                                            if "\n" in toAppend:
                                                                    toAppend = toAppend.replace("\n", "")
                                                            else:
                                                                    toAppend = text[:x]
                                                            lista.append(toAppend)
                                                            latestEnter = x
                                        print(lista)

                                    myText = tkinter.Text(window, height="5", width="20", wrap=tkinter.WORD)
                                    myText.pack()
                                    myButton = tkinter.Button(window, text="print", command=get)
                                    myButton.pack()
                                    window.mainloop()

Icons

Ikonę można dodać do swojego okienka (będzie się wyświetlać w lewym, górnym rogu) lub na przycisk.


                                    import tkinter
                                    window = tkinter.Tk()
                                    logo = Photoimage(file="wonsz.png")
                                    window.iconphoto(False, logo)
                                    window.mainloop()
                                    

                                    import tkinter
                                    window = tkinter.Tk()
                                    logo = Photoimage(file="wonsz.png")
                                    Button(window, image=logo).pack()
                                    window.mainloop()

Colorchooser

Colorchooser to paleta RGB. Po wybraniu koloru na palecie i kliknięciu "OK", wyświetli się wybrany kolor w zapisie RGB i heksadecymalnym.


                                    from tkinter import *
                                    from tkinter import colorchooser
                                    window = Tk()
                                    print(colorchooser.askcolor(title="Wybierz kolor"))
                                    window.mainloop()

Tabs

Tabs pozwala na tworzenie zakładek podobnych do tych, które są w przeglądarce internetowej.


                                    import tkinter
                                    from tkinter import ttk
                                    window = tkinter.Tk()
                                    tabControl = ttk.Notebook(window)
                                    tab1 = tkinter.Frame(tabControl)
                                    tab2 = tkinter.Frame(tabControl)
                                    tabControl.add(tab1, text="Tab 1")
                                    tabControl.add(tab2, text="Tab 2")
                                    tabControl.pack()
                                    tkinter.Label(tab1, text="Tab 1").pack()
                                    tkinter.Label(tab2, text="Tab 2").pack()
                                    window.mainloop()
