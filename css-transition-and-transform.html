Transition and Transform

transform - zmiana stanu, wyglądu, geometrii elementu.

transition - płynne przejście (animacja) pomiędzy dwoma stanami elementu w CSS.

transform: scale(0.9); - zmniejszy obraz (podstawowy rozmiar to 1). Zapisujemy w stylu :hover.

transition: transform 0.2s ease-in-out; - ustawi płynność przejścia transform. Zamiast transform możemy napisać all i wtedy zmieni się nie tylko transform, ale też inne opcje umieszczone w :hover. transition nie zapisujemy w :hover, tylko w zwykłym stylu. ease-in-out to sposób animacji. Zamiast 0.2s można też napisać .2s.


									<style>
											.photo1
											{
													width: 250px;
													transition: all 0.2s ease-in-out;
											}
											.photo1:hover
											{
													-webkit-transform: scale(0.9);
													-ms-transform: scale(0.9);
													transform: scale(0.9);
											} 
									</style>
									
									<img src="img75.jpg" alt="Img" class="photo1">

(Nie działa to na urządzeniach mobilnych.)

transform: rotate(5deg); - obróci obiekt o pięć stopni.

transform: scale(0.95) rotate(5deg); - argumenty transform można łączyć (deg = degree).

transform: skew(0deg, 0deg); - ustawi skos obrazu.

Możliwe właściwości transform: scale(), scaleX(), scaleY(), scale3d(), scaleZ(), rotate(), rotateX(), rotateY(), rotateZ(), rotate3d(), translate(), translateX(), translateY(), translateZ(), translate3d(), skew(), skewX(), skewY(), matrix(), matrix3d() i perspective().

Możliwe właściwości transition: ease, linear, ease-in, ease-out, step-start, step-end, steps() i cubic-bezier().

overflow: hidden; - sprawi, że wielkość elementu nie zmieni się nawet po użyciu transform (nie zmieni się rozmiar zewnętrzny, tylko wewnętrzny).
