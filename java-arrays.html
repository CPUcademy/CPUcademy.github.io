Arrays

We use arrays to store lots of objects of one type. We create them using this pattern: "type of variables that will be stored in this array - name = 'new' - the same type as before - size;" (int[] a = new int[5];). When we create an array, an appropriate amount of memory "in a row" is reserved (the amount required for a given data type times the size), but the values that were there don't vanish. To set them as zeros, we can write: int[] a = {0};. We can also assign all the values at the beginning: int[] a = {1, 2};. An array cannot contain elements that have different data types.


int[] a = new int[4];
a[0] = 1;
a[1] = 2;
a[2] = 3;
a[3] = 4;

for(int i: a) // enhanced for (where to save the values : from where to read the values)
	System.out.println(i);

We can create multidimensional arrays (an array in an array, etc.) A two-dimensional array is called a matrix.


int[] a = new int[4]; // container for 4 variables
 
int[][] a1 = new int[4][3]; // 4 containers (0, 1, 2, 3) that can store 3 variables (0, 1, 2). Overall, it can store 4*3 = 12 variables.
a1[0][1] = 5;
System.out.println(a1[0][1]);

int[][][] a2 = new int[2][3][2]; // 2 containers that can store 3 other containers, which can store 2 variables. Overall, it can store 2*3*2 = 12 variables.
a2[1][2][1] = 6;
System.out.println(a2[1][2][1]);

We can define the array's elements in a loop. We could also use this technique to display all its elements in the console.


int[] a = new int[100];
for(int i = 0; i < 100; i++)
{
	a[i] = i;
	System.out.println("a[" + i + "] = " + a[i]);
}
