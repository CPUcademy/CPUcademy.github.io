Introduction

Java is a widely used, cross-platform programming language known for its portability and robustness. It enables developers to write code that can run on any device with a Java Virtual Machine (JVM), making it highly versatile. Java is extensively used in web applications, enterprise software, mobile applications (especially Android), and large systems development. It supports object-oriented programming. Java is known for its “write once, run anywhere” capability, which means compiled Java code can run on all platforms that support Java without recompilation. Java is a compiled and interpreted, high-level, and general-purpose language. It has a rich set of APIs and a large ecosystem of libraries and frameworks.

IntelliJ IDEA is the most used IDE for coding in Java, and you can download it from this website (choose the free Community Edition). If you want IntelliJ to work, you must install a JDK (Java Development Kit). In IntelliJ, we work on one window. The program results print at the bottom of the screen after you click Run / Run. To create a new file, click File / New File (we type the file name in there), and to create a new project (it can consist of many .py files), click File / New Project. If you have closed a project file and want to open it, click on the tab with the project name (on the left side) and unfold it. Every project and its files are here: C:\Users\[Username]\IdeaProjects\.
