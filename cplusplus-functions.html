Functions

Function is a sequence of program instructions that performs a specific task, packaged as a unit. When we create a function, we can name the arguments it takes (but we don't have to). Arguments are variables that we give to the function so that it can operate on them (generally, a function cannot access objects created outside it in the main() or any other function due to variable scopes, which I will discuss in a moment).

A function can return objects (variables, numbers, arithmetic operations, etc.). When it does, the line that calls a function is equal to the object it returns. To see what the function returned, we have to print the "calling" of it in the console (using cout). We create functions with this pattern: "type of the object it returns - name - parenthesis" (int func()), and below it, we write the content in braces. If the function doesn't return anything, we can use the void type. If we want to return from a void function, we can use: return; (an empty return). It can be useful when we want to stop such a function.

We have to write all functions above the main() function. To call a function (run it), we write its name and a parenthesis (we type a semicolon at the end only if this line isn't, e.g., inside a cout, but stands alone). If it takes any arguments, we write them inside the parenthesis. The main() function can also take arguments, which is used when executing programs from CMD or creating projects (I will talk about it in the upcoming lessons).


#include <iostream>
using namespace std;

void function()
{
	cout << "Function" << endl;
}

int addition(int x, int y) // this function will return an int object and take two int variables as arguments
{
	cout << "Addition" << endl;
	return x + y;
}

int inline_addition(int x, int y){return x + y;} // inline function (intended for short operations)

int main()
{
	function();

	int x = 2, y = 4;
	cout << addition(x, y) << endl;
	cout << inline_addition(x, y) << endl;

	return 0;
}

In C++, we can declare a function at the beginning of the page before the main() function, e.g., void function(int, int);, and define it at the bottom after the main() function: void function(int x, int y){...}. Some people do it to organize their code.

Variable scopes

Not every variable is accessible in every part of the program. Variables created in the main block (outside of any functions), are called global variables. They are accessible in all functions and instructions. Variables created between braces (inside a loop, conditional statement, function, etc.) are accessible only inside these braces and are called local variables. The default value of global variables is 0, but local variables do not have one. When initializing a local variable, it's worth adding an initial value (if we do not input it from the user).

Now that we know what variable scopes are, we can see that in the function example, even if we change the values of the x and y variables in the addition() function, their values in the main() function will not change. These variables, when in the addition() function, are local variables whose values have been imported as arguments and are then returned to the main() function.


#include <iostream>
using namespace std;

int global_variable = 5;

void func()
{
	cout << global_variable << endl;
}

void func2(int local_variable)
{
	cout << local_variable << endl;
}

int main()
{
	func();
	global_variable = 6;
	func();

	int local_variable = 7;
	func2(local_variable); // it wouldn't work without giving the variable as an argument

	return 0;
}

Function overloading (polymorphism)

Two functions can have the same name as long as they have a different number of arguments and / or they have different types. This behavior is called function overloading, or in other words - polymorphism. If we give two double arguments, the function that takes two variables of this type will execute.


#include <iostream>
using namespace std;

int addition(int x, int y)
{
	return x + y;
}

double addition(double x, double y)
{
	return x + y;
}

int main()
{  
	cout << addition(5, 6) << endl;
	cout << addition(5.5, 6.6) << endl;
	return 0;
}

Recursive functions

A recursive function is a function that calls itself. Recursion is often used in various algorithms.


#include <iostream>
using namespace std;

void function()
{
	cout << "x" << endl;
	function();
}

int main()
{    
	function();
	return 0;
}
