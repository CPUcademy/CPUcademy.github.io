Element positioning and Flexbox
    display: block; Makes the element a block-level element. If display: none; was used, this restores its visibility. display: inline-block; An alternative to float. float: left; Allows elements to appear side by side. Variants include right, inherit, and none (default). clear: both; Clears the float effect, commonly used like this: <div style="clear: both;"></div>. justify-content Aligning flex items along the main axis (flex-start, flex-end, center, space-between, space-around, space-evenly). z-index Setting the stack order of an element. Higher values place the element on top. z-index only works on positioned elements (relative, absolute, fixed, or sticky).

position: fixed; - ustala stałą pozycję elementu na stronie. Przy określaniu położenia można też wykorzystać właściwości top: 0; i left: 0;. Ta komenda ma też argument relative i on ustawi "dziecku" pozycję relatywną do jego "rodzica" w kaskadzie.

flex-wrap: nowrap;

Zasada pozycjonowania divów

- Do wszystkich bloków ułożonych szeregowo (czyli obok siebie) dodajemy zapis CSS: float: left;.

- Kiedy już nie chcemy ustawiać ich szeregowo, tworzymy pusty <div> ze stylem: clear: both; (<div style="clear: both;"></div>).
