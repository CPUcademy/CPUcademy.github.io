Element positioning and Flexbox

float and clear

The float property is used to position (stick) elements to the left or right of their container, allowing other objects to wrap around them. The clear property clears the float effect.

Float options

float: left; Floating the element to the left.
float: right; Floating the element to the right.
float: none; The default value (the element does not float).

Clear options

clear: left; Clearing the float effect from the left.
clear: right; Clearing the float effect from the right.
clear: both; Clearing the float effect from both sides.
clear: none; The default value (the effect is not cleared). 

<style>
    .float-left 
    {
        float: left;
        width: 50%;
        background-color: lightblue;
    }
    
    .float-right 
    {
        float: right;
        width: 50%;
        background-color: lightcoral;
    }
</style>

<div class="float-left">Left Float</div>
<div class="float-right">Right Float</div>
<div style="clear: both;">This element clears both floats.</div>

position

The position property specifies the type of positioning method used for an element. It can take several values:

Position options

position: static; The default positioning; elements are positioned according to the normal flow of the document.
position: relative; Positioning the element relative to its normal position.
position: absolute; Positioning the element relative to its nearest positioned ancestor (not static).
position: fixed; Positioning the element relative to the viewport, meaning it stays in the same place even when scrolling.
position: sticky; A hybrid of relative and fixed; the element is treated as relative until it crosses a specified threshold, then it becomes fixed. 

<style>
    .relative {
        position: relative;
        top: 10px;
        left: 10px;
        background-color: lightgreen;
    }
    .absolute {
        position: absolute;
        top: 50px;
        left: 50px;
        background-color: lightyellow;
    }
    .fixed {
        position: fixed;
        top: 0;
        right: 0;
        background-color: lightpink;
    }
</style>

<div class="relative">Relative Positioning</div>
<div class="absolute">Absolute Positioning</div>
<div class="fixed">Fixed Positioning</div>

display

The display property specifies the display behavior of an element. Here are some common values:

Display options

display: block; Displaying the element as a block (takes up the full width).
display: inline; Displaying the element as an inline element (only takes up as much width as necessary).
display: inline-block; Displaying the element as an inline-level block container, allowing width and height to be set.
display: none; Not displaying the element at all (it will not take up any space).
display: flex; Becoming a flex container, enabling the use of Flexbox properties.
display: grid; Becoming a grid container, enabling the use of CSS Grid properties. 

<style>
    .block {
        display: block;
        background-color: lightblue;
        margin: 10px 0;
    }
    .inline {
        display: inline;
        background-color: lightcoral;
        margin: 5px;
    }
    .inline-block {
        display: inline-block;
        width: 100px;
        height: 50px;
        background-color: lightgreen;
        margin: 5px;
    }
    .none {
        display: none;
    }
</style>

<div class="block">Block Element</div>
<span class="inline">Inline Element</span>
<span class="inline">Another Inline Element</span>
<div class="inline-block">Inline Block</div>
<div class="none">This element is not displayed.</div>

Flexbox

Flexbox is a layout model that allows for the efficient arrangement of elements within a container. It provides more control over alignment, direction, and spacing of items.

Flexbox properties

display Defining a flex container (flex, inline-flex, block, inline, inline-block, none).
position Defining the positioning method for an element (static, relative, absolute, fixed, sticky).
flex-direction Defining the direction of the flex items (row, row-reverse, column, column-reverse).
justify-content Aligning flex items along the main axis (flex-start, flex-end, center, space-between, space-around).
align-items Aligning flex items along the cross axis (flex-start, flex-end, center, baseline, stretch).
flex-wrap Controlling whether flex items should wrap onto multiple lines (nowrap, wrap, wrap-reverse).
flex-grow Defining the ability for a flex item to grow if necessary (e.g., 2).
flex-shrink Defining the ability for a flex item to shrink if necessary (e.g., 2).
flex-basis Defining the default size of an element before the remaining space is distributed (e.g., 2).
top, left, right, bottom Specifying the distance between an object and the given screen side (e.g., 0). This requires setting the element to position absolute or relative.
z-index Setting the stack order of an element. Higher values place the element on top. z-index only works on positioned elements (relative, absolute, fixed, or sticky).
