Conditional statements

if conditional statement

The if conditional statement allows us to check if a given condition is fulfilled. We commonly use logical operators in conditional statements.


int x = 5;
if(x == 5)
	System.out.println("The x variable equals 5.");

We also very often use the modulo operator in this instruction. In the example below, the instruction says that if the x variable is divisible by 2, it is even.


int x = 6;
if(x % 2 == 0)
	System.out.println("The x variable is an even number.");

With this instruction, we can make use of boolean variables. We can just write if(x), and if x is true, the instruction will be executed.

else and else if conditional statements

else is an instruction that lets us simplify our code by not writing another if but replacing it.


int x = 5;
if(x > 0)
	System.out.println("The x variable is a positive number.");
else
	System.out.println("The x variable is a negative number.");

The code from the example above isn't accurate because we could type in 0. In this case, we need else if. It is used the same way as else, but we type a condition in it (like in an if). We can use as many of them as we want, relating to a single if.


int x = 5;
if(x > 0)
	System.out.println("The x variable is a positive number.");
else if(x < 0)
	System.out.println("The x variable is a negative number.");
else
	System.out.println("The x variable equals 0.");

You may wonder why we even need else if if we type in a condition exactly like in an if. It is because it makes the code simpler to read. Also, if any of the conditions has been met, the rest won't be considered, which wouldn't be the case if we wrote multiple if statements.

break and continue keywords

The break keyword is used to break the continuity of a loop, and continue can skip iterations within it.


int x = 0;
while(true)
{
	if(x == 5)
		break;
	System.out.println(x);
	x++;
}

for(int y = 1; y <= 10; y++)
{
	if(y == 4 || y == 8)
		continue;
	System.out.println("y equals: " + y);
}

switch conditional statement

switch is an instruction that simplifies complex conditions with if and else if statements.


int x = 5;

switch(x)
{
	case 0:
		System.out.println("x equals 0");
		break;
	case 25:
		System.out.println("x equals 25");
		break;
	case 50:
		System.out.println("x equals 50");
		break;
	default:
		System.out.println("x doesn't equal 0, 25, or 50");
}

An argument of this statement or the case argument can be an int a char (it will convert characters to numbers based on the ASCII array).

If we want several cases to be called upon in a given case, we do not put the break statement at the end of it (they will execute until a break is encountered or the switch instruction ends).

Instructions with ?

The ? sign allows us to build less complicated conditions. If the condition is true, the instruction on the left side of the colon will execute, and if not - the one on the right side. Check the results in your editor.


System.out.println(7 > 4 ? 4: 7);

int a = 7, b = 10;
String x = a >= b ? "a >= b": "a <= b";

System.out.println(x);
