Tkinter basics

Tworzenie okna i nadawanie mu tytułu


                                    import tkinter
                                    window = tkinter.Tk()
                                    window.title("Okienko")
                                    window.mainloop()

Komenda Tk() tworzy okno, title("") nadaje mu tytuł, a mainloop() zapobiega temu, żeby okienko zamykało się od razu po uruchomieniu (można jej użyć pisząc po prostu mainloop() lub nazwaOkienka.mainloop()). W programie wystarczy jedna komenda mainloop(), nawet jeśli piszemy program działający na kilku oknach.

Komenda geometry()

Komenda geometry() ustawia wielkość okienka w pikselach. Pierwsza wartość to szerokość, a druga - wysokość.


                                    import tkinter
                                    window = tkinter.Tk()
                                    window.geometry("300x200")
                                    window.mainloop()

Do tej komendy można dodać też dodatkowe liczby definiujące, w którym miejscu ekranu pojawi się okienko: window.geometry("300x200+100+200").

Komenda pack()

Komenda pack() pozwala na załadowanie widżetu do okienka.


                                    import tkinter
                                    window = tkinter.Tk()
                                    tkinter.Label(window, text="Wonsz").pack()
                                    window.mainloop()

Więcej o komendzie Label na lekcji drugiej!

Ten kod zrobi to samo, co ten powyżej.


                                    import tkinter
                                    window = tkinter.Tk()
                                    myLabel = tkinter.Label(window, text="Wonsz")
                                    myLabel.pack()
                                    window.mainloop()

Do komendy pack() możemy dodać argument side, który określi stronę okienka, po której znajdzie się dany widżet.


                                    import tkinter
                                    window = tkinter.Tk()
                                    myLabel = tkinter.Label(window, text="Wonsz 1")
                                    myLabel.pack(side=tkinter.RIGHT)
                                    myLabe2 = tkinter.Label(window, text="Wonsz 2")
                                    myLabe2.pack(side=tkinter.LEFT)
                                    window.mainloop()

Komenda place()

Komendą place() możemy ustawić współrzędne widżetu w okienku.


                                    import tkinter
                                    window = tkinter.Tk()
                                    myLabel = tkinter.Label(window, text="Wonsz")
                                    myLabel.pack()
                                    myLabel.place(x=11, y=20)
                                    window.mainloop()

Inna wersja tej komendy wygląda tak:


                                    import tkinter
                                    window = tkinter.Tk()
                                    myLabel = tkinter.Label(window, text="Wonsz")
                                    myLabel.pack()
                                    myLabel.place(relx=0.2, rely=0.3)
                                    window.mainloop()

Komenda destroy()

Komendą destroy() możemy niszczyć okienka i inne widżety: window.destroy().

Można też zastosować inną odmianę tej komendy, która zniszczy okineko po podanym czasie (1000 - 1 sekunda):


                                    import tkinter
                                    window = tkinter.Tk()
                                    window.after(3000, window.destroy)
                                    window.mainloop()

Komenda grid()

Komenda grid() tworzy siatkę podobną do siatki z Bootstrapa. Dodaje się do niej widżety, ustawiając je w wiersze (rows) i kolumny (columns).


                                    import tkinter
                                    window = tkinter.Tk()

                                    myLabel1 = tkinter.Label(window, text="Pierwszy:")
                                    myLabel2 = tkinter.Label(window, text="Drugi:")

                                    myLabel1.grid(row=0, column=0, pady=2)
                                    myLabel2.grid(row=1, column=0, pady=2)

                                    myEntry1 = tkinter.Entry(window)
                                    myEntry2 = tkinter.Entry(window)

                                    myEntry1.grid(row=0, column=1, pady=2)
                                    myEntry2.grid(row=1, column=1, pady=2)

                                    window.mainloop()
