Arrays

We use arrays to store lots of objects of one type. We create them using this pattern: "type of variables that will be stored in this array - name - size;" (int a[4];). To get to the address of the first element of an array, we can write only the name (instead of a[0]). When we create an array, an appropriate amount of memory "in a row" is reserved (the amount required for a given data type times the size), but the values that were there don't vanish. To set them as zeros, we can write: int a[4] = {0};. We can also assign all the values at the beginning: int tab3[4] = {1, 2, 3, 4}; or int tab3[] = {1, 2, 3, 4};. An array cannot contain elements that have different data types.


int a[4];
a[0] = 1;
a[1] = 2;
a[2] = 3;
a[3] = 4;

cout << "a[0] equals " << a[0] << " and has the address " << &a[0] << endl;
cout << "a[1] equals " << a[1] << " and has the address " << &a[1] << endl;
cout << "a[2] equals " << a[2] << " and has the address " << &a[2] << endl;
cout << "a[3] equals " << a[3] << " and has the address " << &a[3] << endl;

for(int i = 0; i < 4; i++)
	cout << a[i] << endl;

We can create multidimensional arrays (an array in an array, etc.) A two-dimensional array is called a matrix.


int a[4]; // container for 4 variables
 
int a1[4][3]; // 4 containers (0, 1, 2, 3) that can store 3 variables (0, 1, 2). Overall, it can store 4*3 = 12 variables.
a1[0][1] = 5;
cout << a1[0][1] << endl;

int a2[2][3][2]; // 2 containers that can store 3 other containers, which can store 2 variables. Overall, it can store 2*3*2 = 12 variables.
a2[1][2][1] = 6;
cout << a2[1][2][1] << endl;

We can define the array's elements in a loop. We could also use this technique to display all its elements in the console.


int a[100];
for(int i = 0; i < 100; i++)
{
	a[i] = i;
	cout << "a[" << i << "] = " << a[i] << endl;
}
