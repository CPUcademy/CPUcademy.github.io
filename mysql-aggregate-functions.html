Aggregate functions

LIMIT clause

Specifying a limit on the number of records that will be displayed as a query result.


SELECT column_name 
FROM table_name 
LIMIT 5;

COUNT function

Counting data records in a table.


SELECT COUNT(*)
FROM table_name;

SUM function

Summing values.


SELECT SUM(column_name)
FROM table_name;

AVG function

Calculating the average value.


SELECT AVG(column_name)
FROM table_name;

ROUND function

Rounding the given value to a given number of decimal places.


SELECT column2_name, ROUND(column2_name, 2)
FROM table_name;

GROUP BY clause

Grouping records by selected values.


SELECT column_name, COUNT(*)
FROM table_name
GROUP BY column_name;

Column references

Referring to columns by their order in the SELECT clause. 1 - column1_name, 2 - column2_name.


SELECT column1_name, column2_name
FROM table_name
GROUP BY 2
ORDER BY 1;

HAVING clause

Adding a condition to the GROUP BY command.


SELECT column_name, COUNT(*)
FROM table_name
GROUP BY column_name
HAVING COUNT(*) < 3;

MAX function

Finding the highest value.


SELECT MAX(column_name)
FROM table_name;

MIN function

Finding the lowest value.


SELECT MIN(column_name)
FROM table_name;

Subqueries with WHERE

MAX statement with WHERE subquery - displaying only the name of the part with the highest price.


SELECT column1_name
FROM table_name
WHERE column2_name = (SELECT MAX(column2_name) FROM table_name);
