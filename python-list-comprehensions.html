List comprehensions

There are three ways of saving ten tens to a list.


list1 = []
for x in range(10):
    list1.append(10)
print(list1)
                                    

list2 = [10] * 10
print(list2)
                                    

list3 = [10 for x in range(10)]
print(list3)

List comprehensions provide a simple way to create lists based on a sequence we can iterate (e.g., a string). The object the comprehension returns can be a different list comprehension, a mathematical expression, or a string.


list1 = [letter for letter in "abcd"] # dividing a string into a list of characters
print(list1)

list2 = [int(character) for character in "0123456"] # type casting every string character
print(list2)

x, y, z = [int(x) for x in input("Enter three numbers separated by spaces: ").split()] # using the split() command with a list comprehension
print(x + y + z)

list3 = [["a", "b"] for x in range(10)] # creating a two-dimensional list
print(list3)

list4 = [x.lower() for x in "ABCDE"] # turning all letters to lowecase
print(list4)

l = [{"name": "x", "value": 1}, {"name": "y", "value": 2}]
list5 = [x["name"] for x in l]
print(list5)
                                    

squares = [x * x for x in range(5)]
print(squares)                                        

list1 = [x * y for x, y in ([1, 2], [3, 4])]
print(list1)

list2 = [[x for x in range(3)] for x in range(10)]
print(list2)

list3 = [x * y for x, y in ([a, a + 1] for a in range(10))] # x = a, y = a + 1 for each "a" in range(10); 10 iterations overall
print(list3)

All data structures can be filled with data using list comprehensions.


set1 = {x * x for x in range(10)}
print(set1)

dict1 = {x: x * x for x in range(10)}
print(dict1)

List comprehensions can contain conditions.


list1 = [x * x for x in range(11) if x % 2 == 0]
print(list1)

text = "one: 1, two: 2, three: 3, four: 4"
list2 = [int(character) for character in text if character.isdigit()]
print(list2)
