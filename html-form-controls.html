Form controls

The <form> tag is used to create interactive controls that users can use to enter data and submit to a web server for processing. It is a fundamental part of web development, allowing users to interact with a website, such as filling out contact forms, login forms, and surveys. A basic form structure looks like this:


<form action="submit_form.php" method="post">
    <!-- Form elements go here -->
</form>

The action attribute specifies the URL to which the form data is sent. The method attribute defines how data is sent. Common values include post (to send data in the body of the request) and get (to append data to the URL).

Form Input Types

The different values of the type attribute in the <input> tag:

  • text
    • Defines a single-line text input field for user input.
    • <input type="text" name="username" placeholder="Enter your username">
    • Instead of placeholder, we can use value. Check the difference by trying to enter some text.
  • password
    • Defines a password input field. Characters typed are obscured.
    • <input type="password" name="password" placeholder="Enter your password">
  • email
    • Defines a field for entering an email address. It can validate email format automatically (if contains @, etc.).
    • <input type="email" name="email" placeholder="Enter your email">
  • number
    • Defines a field for numeric input. You can set a range using the min and max attributes. An optional step attribute defines the increments between valid values (e.g., step="0.1" for decimal values).
    • <input type="number" name="age" min="18" max="100" placeholder="Enter your age">
  • checkbox
    • Defines a checkbox, allowing users to select one or more options.
    • <input type="checkbox" name="subscribe" value="yes"> Subscribe to newsletter
    • Subscribe to newsletter
  • radio
    • Defines a radio button. It allows users to select one option from a group.
    • <input type="radio" name="gender" value="male"> Male
    • <input type="radio" name="gender" value="female"> Female
    • Male
    • Female
  • file
    • Defines a file upload button that allows users to select and upload a file.
    • <input type="file" name="profile_pic">
  • submit
    • Defines a button for submitting the form data.
    • <input type="submit" value="Submit">
  • reset
    • Defines a button that resets all form fields to their default values.
    • <input type="reset" value="Reset">
  • button
    • Defines a generic button for use with JavaScript (often for custom form actions).
    • <input type="button" value="Click me" onclick="alert('Button clicked!')">
    • This version works the same. If we do not add the type="button", it will work like the submit button.
    • <button type="button" onclick="alert('Button clicked!')">Click me</button>
  • textarea
    • Defines a multi-line text input field, often used for comments or messages. To block resizing (in CSS): resize: none;.
    • <textarea name="message" rows="4" cols="10" placeholder="Enter your message"></textarea>
  • select <option>
    • Defines a drop-down menu for selecting one or more options.
    • 
<select name="country">
    <option value="USA">USA</option>
    <option value="Canada">Canada</option>
    <option value="UK">UK</option>
</select>
