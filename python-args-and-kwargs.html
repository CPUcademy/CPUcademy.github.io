*args and **kwargs

When we don't know the number of arguments we will give to a method or a function, we can use *args and **kwargs. We do it by putting * or ** before the name of an argument (it can have any name).

*args

*args - non-keyword (normal) arguments. A keyword argument is an argument that contains a key-value pair.


def function(*numbers):
    sum1 = 0
    for x in numbers:
        sum1 += x
    print("Sum: ", sum1)

function(1, 2)
function(1, 2, 3)
function(1, 2, 3, 4)

**kwargs

**kwargs - keyword arguments. Here, we can add objects that have values assigned to them (like in a dictionary).


def function(**data):
    for key, value in data.items():
        print(f"{key}: {value}")

function(name = "John", surname = "Anderson", age = 20)
function(name = "Liam", surname = "Brown", age = 20, country = "USA")

*args and **kwargs

While using *args and **kwargs combined with standard arguments, remember to use this order: standard arguments, *args, **kwargs.


def function(argument1, argument2, *args, **kwargs):
    print("Standard argument nr. 1:", argument1)
    print("Standard argument nr. 2:", argument2)
    for x in args:
        print("Args:", x)
    for key, value in kwargs.items():
        print(f"Kwargs: {key} - {value}")

function(1, 2, 3, 4, a = 5, b = 6, c = 7)

* and ** symbols

A single asterisk can also be used to convert a list to a string inside the print() command.


list1 = ["one", 2]
print(*list1)
print(*list1, sep = ", ") # separating each element with a comma

Two asterisks can also be used to link (merge) dictionaries.


dict1 = {"one": 1}
dict2 = {"two": 2}
linked = {**dict1, **dict2}
print(linked)
