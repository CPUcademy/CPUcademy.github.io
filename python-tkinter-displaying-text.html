Displaying text

Komenda Label służy do wyświetlania tekstu. Dla tej komendy można zdefiniować dużo argumentów.


                                    import tkinter
                                    window = tkinter.Tk()
                                    myLabel = tkinter.Label(window, text="Wonsz")
                                    myLabel.pack()
                                    window.mainloop()

Argument szerokości - width

Ten argument definiuje szerokość tekstu (ten tekst jest na całą szerokość okienka, mimo że tego nie widać).


                                    import tkinter
                                    window = tkinter.Tk()
                                    myLabel = tkinter.Label(window, text="Wonsz", width=40)
                                    myLabel.pack()
                                    window.mainloop()

Argument wysokości - height

Ten argument definiuje wysokość tekstu (ten tekst jest na całą wysokość okna, mimo że tego nie widać).


                                    import tkinter
                                    window = tkinter.Tk()
                                    myLabel = tkinter.Label(window, text="Wonsz", height=20)
                                    myLabel.pack()
                                    window.mainloop()

Kolor tekstu - fg

Argument fg (ang. foreground) definiuje kolor tekstu.


                                    import tkinter
                                    window = tkinter.Tk()
                                    myLabel = tkinter.Label(window, text="Wonsz", fg="red")
                                    myLabel.pack()
                                    window.mainloop()

Kolor tła - bg

Argument bg (ang. background) definiuje kolor tła tekstu.


                                    import tkinter
                                    window = tkinter.Tk()
                                    myLabel = tkinter.Label(window, text="Wonsz", bg="red")
                                    myLabel.pack()
                                    window.mainloop()

Czcionka - font

Argument font definiuje rozmiar i krój czcionki.


                                    import tkinter
                                    window = tkinter.Tk()
                                    myLabel = tkinter.Label(window, text="Wonsz", font=("Arial", 20))
                                    myLabel.pack()
                                    window.mainloop()

Miejsce tekstu w okienku - anchor

Argument anchor definiuje stronę świata, po której znajduje się tekst względem swoich rozmiarów. Po zdefiniowaniu argumentu width, tekst ma określoną szerokość, której nie widać, jeśli tekstu jest za mało - wtedy możemy ustawić ten tekst po którejś stronie jego "niewidzialnej ramki". e oznacza "east" czyli wschód, co oznacza, że tekst znajdzie się po prawej stronie.


                                    import tkinter
                                    window = tkinter.Tk()
                                    myLabel = tkinter.Label(window, text="Wonsz", anchor="e", width=60)
                                    myLabel.pack()
                                    window.mainloop()
